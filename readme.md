## Installation

Clone the project

```bash
  git clone git@github.com:RemySd/morpion-symfony.git
  cd morpion-symfony
```

Charger composer

```bash
  composer install
```

## Deployment

To deploy this project run

```bash
  symfony serve
```

Go to https://127.0.0.1:8000
